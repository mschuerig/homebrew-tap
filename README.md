# homebrew-tap

Personal Homebrew tap.

## Install

```sh
brew tap mschuerig/tap
brew install mschuerig/tap/anythingllm-feeder
```

## Formulae

- **anythingllm-feeder** — extract documents/videos to Markdown with `forage`
  and feed them into a local AnythingLLM with `ingest`. macOS Apple Silicon
  only. See [`Formula/anythingllm-feeder.rb`](Formula/anythingllm-feeder.rb)
  for prerequisites and the release workflow.
